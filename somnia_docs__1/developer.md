# Network Info

**The Somnia Mainnet is LIVE**

**Developers who deploy Smart Contracts on Mainnet require Somnia Tokens, SOMI. It is a real-world utility token that can be purchased on a list of CEXs and DEXs**[**here**](/get-started/getting-started-for-mainnet#get-somi-tokens)**.**

**Developers who are deploying Smart Contracts and need Somnia Test Tokens, STT. Please join the**[**Discord**](https://discord.com/invite/somnia)**. Go to the #dev-chat channel, tag the Somnia DevRel,**`**@emma_odia**`**and request Test Tokens. You can also use the Faucet:**[**https://testnet.somnia.network/**](https://testnet.somnia.network/)**You can also email**`**[[email protected]](/cdn-cgi/l/email-protection)**`**with a brief description of what you are building and your GitHub profile.**
