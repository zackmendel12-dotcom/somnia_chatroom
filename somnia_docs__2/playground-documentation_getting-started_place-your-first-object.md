# Place your first object

Go to the playground from the Somnia betanet page or go to <https://playground.somnia.network/>[](https://playground.somnia.network/)

![](https://docs.somnia.network/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FkC3S4jjM9eI5TkJDPhBZ%2Fblobs%2F2xWPORbJvQoiHqrRRoXD%2Fimage.png&width=768&dpr=4&quality=100&sign=e71ee34&sv=2)

Click the world you created using the [create your first world guide](/playground-documentation/getting-started/create-your-first-world). Then click the jump-in button.

![](https://docs.somnia.network/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FkC3S4jjM9eI5TkJDPhBZ%2Fblobs%2FeKjo0gZGUDJkz2GhJCGf%2Fimage.png&width=768&dpr=4&quality=100&sign=36095ca9&sv=2)

Select the items menu in the top right and then the item list button.

![](https://docs.somnia.network/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FkC3S4jjM9eI5TkJDPhBZ%2Fblobs%2FX3m4jMAKV344fqdXiJcU%2Fimage.png&width=768&dpr=4&quality=100&sign=57359eab&sv=2)

Here, you can select an item from the list you want to place. Then click the place button.

![](https://docs.somnia.network/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FkC3S4jjM9eI5TkJDPhBZ%2Fblobs%2FdIkzuUjsRYtxyD6MqZVS%2Fimage.png&width=768&dpr=4&quality=100&sign=99912605&sv=2)

Here we have placed the Somnia box into the playground! You can edits it's size and position by going to the items in world list, selecting edit on the item which will allow you to move and resize it.

![](https://docs.somnia.network/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FkC3S4jjM9eI5TkJDPhBZ%2Fblobs%2Fo0J1LdOz8afEA9fh7gBg%2Fimage.png&width=768&dpr=4&quality=100&sign=d9f8a527&sv=2)

You can place many items and customize your playground however you want! These items will persist between sessions, so when you log in, they will still be there. Learn how you can make your own custom objects with the [Create First Object guide](/playground-documentation/getting-started/create-your-first-object).
