# Item Gallery

You can find the item galery by going to the item builder page and clicking on item gallery:

![](https://docs.somnia.network/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FkC3S4jjM9eI5TkJDPhBZ%2Fblobs%2FXkfEMWNllGXu6RpTO4XS%2Fimage.png&width=768&dpr=4&quality=100&sign=89ef39f9&sv=2)

This shows all the items created by users of the Somnia network. If you like any of them, you can click mint. This will create an NFT in your wallet, which you can use in the Playground (see Create your first object for a guide on how to do this).

![](https://docs.somnia.network/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FkC3S4jjM9eI5TkJDPhBZ%2Fblobs%2FivK43CDIzn3g927PlZSm%2Fimage.png&width=768&dpr=4&quality=100&sign=1163fa70&sv=2)
